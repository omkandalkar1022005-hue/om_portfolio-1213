## ✨ **Features**

### 🎨 **Design Features**
- **Glass-morphism Effect** - Modern transparent cards with blur effect
- **Gradient Colors** - Beautiful gradient backgrounds and text
- **Custom Cursor** - Unique cursor with follower effect
- **Particles Background** - Animated background with particles.js
- **AOS Animations** - Scroll-based animations
- **Floating Elements** - Animated floating shapes and icons
- **Responsive Design** - Perfect on mobile, tablet, and desktop

### 🚀 **Interactive Features**
- **Typed.js Animation** - Dynamic text changing effect
- **Filterable Projects** - Category-based project filtering
- **Skill Modals** - Detailed skill information popups
- **Copy to Clipboard** - One-click contact copying
- **Toast Notifications** - User-friendly alerts
- **Share Button** - Share portfolio on social media
- **Floating Social Sidebar** - Quick access to social profiles
